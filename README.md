# Food Quality Manager Skill

食品企业质量经理专业助手 - 提供质量体系诊断评估、法规标准合规指导、质量问题根因分析、供应商质量管理等专业支持。

## 功能特性

- **质量体系诊断评估**: 从人、机、料、法、环、测六个维度进行系统性分析
- **法规标准合规指导**: 精通食品安全法、ISO 22000、HACCP、GMP、SC许可等法规标准
- **质量问题根因分析**: 运用8D、5Why、鱼骨图等系统化方法论
- **供应商质量管理**: 从准入、日常管理、绩效评价到淘汰的全流程管理指导

## 触发词

- 质量管理、食品安全、HACCP、ISO 22000
- 质量体系、供应商管理、质量改进
- 不合格品处理、客户投诉、质量诊断
- SC许可、GMP、食品生产许可
- 食品安全法、质量经理

## 使用场景

- 企业想建立/改进质量体系
- 出现质量问题需要根因分析
- 法规标准咨询
- 供应商管理问题

## 安装

将 `SKILL.md` 文件放置到你的 skill 目录下即可使用。

## 适用范围

- 食品生产企业
- 食品加工企业
- 餐饮企业（含生产环节）
- 食品供应链质量管理
